# Multiplication Master

A fun and engaging multiplication practice app for children, designed to build confidence and fluency through progressive difficulty levels and real-world rewards.

## Features

- **4 Progressive Levels**: From Starter (5×5) to Champion (timed full table)
- **Kid-Friendly UI**: Large buttons, colorful design, and number pad input
- **Real Rewards**: Screenshot-able vouchers for Percy Jackson reading time
- **Immediate Feedback**: Kids learn from mistakes as they go
- **Timer Challenge**: Level 4 adds time pressure for advanced practice

## Levels

1. **Starter** (🌟): 25 questions, up to 5×5 → 1 min reward
2. **Builder** (🚀): 15 questions, up to 10×10 → 1 min reward
3. **Master** (👑): 10 questions, full table → 2 min reward
4. **Champion** (🏆): 10 questions, full table + 100s timer → 5 min reward

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This app is designed to be deployed on Vercel:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Deploy with one click!

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- React 19

## How It Works

1. Yonatan selects a level
2. Answers multiplication questions using the number pad
3. Gets immediate feedback on each answer
4. Earns a voucher if all answers are correct
5. Screenshots the voucher to claim reading time!
